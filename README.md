# 📊 Data Analytics Project – Codveda Internship

## 📌 Overview
This project is completed as part of the **Data Analytics Internship at Codveda Technology**.  
It includes data preprocessing, analysis, machine learning, and dashboard creation using multiple real-world datasets.

---

## 🎯 Objectives
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Visualization  
- Machine Learning Models  
- Time Series Analysis  
- Clustering (K-Means)  
- Sentiment Analysis (NLP)  
- Dashboard Creation (Tableau/Power BI)  

---

## 🗂️ Datasets Used
- Iris Dataset  
- Stock Prices Dataset  
- Sentiment Dataset  
- House Price Dataset  
- Customer Churn Dataset  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- NLTK / TextBlob  
- Tableau / Power BI  

---

## 🚀 Project Workflow

### 🔹 Level 1: Basic
- Data Cleaning (handling missing values, duplicates)  
- Exploratory Data Analysis (mean, median, correlation)  
- Data Visualization (bar charts, line plots, scatter plots)  

### 🔹 Level 2: Intermediate
- Regression Analysis (Linear Regression)  
- Time Series Analysis (Stock dataset)  
- Clustering (K-Means Algorithm)  

### 🔹 Level 3: Advanced
- Classification Models (Logistic Regression, Decision Tree, Random Forest)  
- Sentiment Analysis using NLP  
- Interactive Dashboard Creation  

---

## 📊 Results
- Cleaned and structured datasets  
- Generated meaningful insights through visualization  
- Built predictive models with good accuracy  
- Developed interactive dashboards
