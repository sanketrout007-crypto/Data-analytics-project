# 📊 Data Analytics Internship Project

## 🔹 Overview

This project is completed as part of the **Data Analytics Internship**.
It includes **three levels (Basic, Intermediate, Advanced)** covering data cleaning, analysis, machine learning, NLP, and dashboard creation.
